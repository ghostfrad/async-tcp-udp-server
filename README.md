# Асинхронный TCP/UDP Сервер

Тестовое задание: асинхронный сервер на C++ с поддержкой TCP и UDP через epoll.

## Функциональность

- Асинхронная обработка подключений через epoll
- Поддержка TCP и UDP протоколов
- Обработка команд: `/time`, `/stats`, `/shutdown`
- Зеркалирование обычных сообщений
- Статистика подключений клиентов
- Systemd сервис для запуска

## Сборка

```bash
make
